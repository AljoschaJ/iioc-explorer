# IIOC Explorer

Interactive search across 6,289 presentations at 20 editions of the IIOC (2006-2026).

- Type a first or last name; autocomplete handles middle-initial variants.
- Click a paper to expand; any name is a link to that person's profile.
- Click a session title to see every paper in that session.
- 'Suggest correction' saves edits to your browser. Use 'Download corrections' to share them back.

## Open

- Online: https://aljoschaj.github.io/iioc-explorer/
- Offline: download index.html and open it.
